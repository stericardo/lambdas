## Summary

The idea is to provide a repo with several functionalities about lambdas

## Clone repository
> git clone https://github.com/stericardo/lambdas.git
> Generate eclipse
> gradle eclipse

## Run configurations

### Build

• lambdas-build
• gradle clean build
• ${workspace_loc:/lambdas}


### Run
• lambdas-run

• gradle clean build

• ${workspace_loc:/lambdas}
